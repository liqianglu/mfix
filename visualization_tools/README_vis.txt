animate  : Folder with codes to visualize data in the MFIX SPx output files

           Linux executables exist for both the new and old animate code.
           The executables might not work on all systems, depending on
           the versions of some shared libraries.  Source code for the
           new version is included.

           A windows exectable of the old code is included.
           

dem-plot : folder to visualize particles using the *.vtp files in the
           DEM part of MFIX

           A Linux executable has been included.  The executable might not 
           work on all systems, depending on the versions of some shared 
           libraries.  Source code is included.
